Описание

Минималистичный стул для интерьеров в стиле сканди. Оптимизированная сетка, готовые материалы. Особенности модели: • Низкополигональная геометрия (2.4K треугольников) • Текстуры 2K: Albedo, Normal, Roughness, Metalness • Корректные UV-развёртки без наложений • Готовые материалы для Blender, Unity, Unreal Engine В комплекте: • Файлы форматов: FBX, OBJ, GLTF, BLEND • Папка с текстурами в PNG • README с инструкцией по импорту • Лицензия: свободное использование с указанием автора
